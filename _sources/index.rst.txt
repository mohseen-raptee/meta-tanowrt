TanoWrt Embedded Linux Distribution
###################################

.. container:: sidebar

   .. image:: sphinx-static/tano-systems.png
      :target: https://tano-systems.com/en.html
      :align: center

TanoWrt is an operating system (OS) distribution based on the Linux kernel
and focused on running in embedded systems, including real-time systems.
